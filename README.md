# 한글 스타일 자동적용

한글의 `.sty` 스타일 파일과 스크립트 매크로를 이용하여 문서의 제목, 본문, 글머리표, 표 관련 스타일을 자동으로 적용하는 방법을 정리한 저장소입니다.

문단 앞에 `[스타일명]` 태그를 입력한 뒤 매크로를 실행하면 태그가 삭제되고, 해당 문단에 연결된 한글 스타일이 적용됩니다.

## 블로그

전체 작업 과정과 사용 방법은 GitHub Pages에서 확인할 수 있습니다.

- [사용 가이드 보기](https://rererererete45.github.io/hwp-style-automation/)

## 주요 기능

- 한글 `.sty` 스타일 파일 분석
- 사용할 스타일과 내부 번호 정리
- 선택한 스타일만 사용한 문서 작성
- 기존 문서에 스타일 태그 부여
- 태그와 스타일 내부 번호를 연결한 매크로 생성
- 태그 삭제 및 문단 스타일 자동 적용
- 제목, 본문, 글머리표, 표 스타일 처리

## 작업 흐름

```text
한글 스타일 파일 만들기
→ .sty 파일 분석
→ 사용할 스타일 선택
→ 스타일 태그가 포함된 문서 작성
→ 스크립트 매크로 생성
→ 한글에서 자동 스타일 적용
```

## 태그 사용 예시

```text
[개요1] 문서 제목
[본문강조] 문서의 핵심 내용을 입력한다.
[개요2] 세부 제목
[본문] 일반 설명을 입력한다.
[본문1] 세부항목을 입력한다.
[표중앙] 표 안의 가운데 정렬 내용
[표왼쪽] 표 안의 왼쪽 정렬 항목
[표내용] 표 안의 세부 내용
```

스타일에 문단 번호나 글머리표가 포함되어 있다면 번호와 기호를 본문에 직접 입력하지 않습니다.

## 사용 시 주의사항

- 스타일 내부 번호는 문서와 `.sty` 파일에 따라 달라질 수 있습니다.
- 예제의 내부 번호를 다른 문서에 그대로 적용하지 마십시오.
- 태그 문자열 뒤에는 공백 한 칸을 유지해야 합니다.
- 매크로는 원본이 아닌 복사본에서 먼저 실행하는 것이 안전합니다.
- 실행 후 제목 단계, 글머리표, 표 정렬과 글꼴을 직접 검수해야 합니다.
- 매크로 보안 수준을 변경한 경우 작업 후 원래 설정으로 복원하십시오.
