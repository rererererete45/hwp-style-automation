# 한글 스타일 파일 기반 문서 작성 및 자동 스타일 적용

사용자가 한글의 `.sty` 스타일 파일을 제공하면 스타일을 분석하고,
선택한 스타일을 활용하여 문서와 한글 스크립트 매크로를 작성합니다.

## 주요 기능

- `.sty` 파일 스타일 목록 분석
- 사용자 지정 스타일만 사용
- 태그 기반 문서 작성
- 한글 스크립트 매크로 생성
- 스타일 두 번 적용

## 사용 절차

### 1단계: 스타일 파일 분석

`.sty` 파일을 업로드하고 스타일 정보를 확인합니다.

### 2단계: 사용할 스타일 선택

사용할 스타일명과 내부 번호를 지정합니다.

### 3단계: 문서 작성

각 문단 앞에 스타일 태그가 포함된 문서를 작성합니다.

### 4단계: 매크로 실행

생성된 한글 스크립트 매크로를 실행하여 스타일을 적용합니다.
