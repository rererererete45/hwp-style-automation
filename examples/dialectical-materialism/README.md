# 변증법적 유물론 문서 예제

변증법적 유물론을 주제로 작성한 문서에 한글 스타일 태그를 적용하는 예제입니다.

## 예제 파일 구성

```text
dialectical-materialism/
├─ README.md
├─ style-list.md
├─ tagged-document.txt
└─ generated-macro.js
```

* `README.md` : 사용 방법 설명
* `style-list.md` : 스타일명과 내부 번호
* `tagged-document.txt` : 태그가 포함된 원본 문서
* `generated-macro.js` : 태그를 처리하는 매크로

## 사용한 스타일

| 내부 번호 | 스타일명      | 태그            |
| ----- | --------- | ------------- |
| 0     | 바탕글       | `[바탕글]`       |
| 1     | 간지 제목     | `[간지 제목]`     |
| 2     | 개요 1      | `[개요 1]`      |
| 3     | 개요 2      | `[개요 2]`      |
| 4     | 개요 3      | `[개요 3]`      |
| 5     | 개요 4      | `[개요 4]`      |
| 6     | 개요 5      | `[개요 5]`      |
| 10    | 본문        | `[본문]`        |
| 11    | 본문 1      | `[본문 1 (-)]`  |
| 12    | 본문강조      | `[본문강조 (○)]`  |
| 13    | 주의 및 참고문헌 | `[주의 및 참고문헌]` |

스타일 내부 번호는 문서마다 달라질 수 있습니다.

## 태그 작성 방식

```text
[스타일명] 문단 내용
```

예시:

```text
[간지 제목] 변증법적 유물론

[개요 1] 변증법적 유물론의 기본 개념

[본문] 변증법적 유물론은 자연과 사회를 물질적 현실에 기초해 이해하는 철학적 관점이다.
```

## 매크로 동작 방식

```text
문서 처음으로 이동
→ 태그 검색
→ 태그 삭제
→ 스타일 적용
→ 같은 스타일 두 번째 적용
→ 다음 태그 검색
```

`styleMap` 예시:

```javascript
var styleMap = [
    { tag: "[바탕글] ", styleId: 0 },
    { tag: "[간지 제목] ", styleId: 1 },

    { tag: "[개요 1] ", styleId: 2 },
    { tag: "[개요 2] ", styleId: 3 },
    { tag: "[개요 3] ", styleId: 4 },
    { tag: "[개요 4] ", styleId: 5 },
    { tag: "[개요 5] ", styleId: 6 },

    { tag: "[본문] ", styleId: 10 },
    { tag: "[본문 1 (-)] ", styleId: 11 },
    { tag: "[본문강조 (○)] ", styleId: 12 },
    { tag: "[주의 및 참고문헌] ", styleId: 13 }
];
```

## 실행 방법

1. `tagged-document.txt` 내용을 복사합니다.
2. 스타일이 등록된 한글 문서에 붙여넣습니다.
3. `generated-macro.js`를 실행합니다.
4. 태그가 제거되고 스타일이 적용되었는지 확인합니다.

## 주의사항

* 태그 뒤 공백까지 `styleMap`에 포함됩니다.
* 스타일 내부 번호는 문서마다 달라질 수 있습니다.
* `.sty` 파일을 변경하면 번호를 다시 확인해야 합니다.
